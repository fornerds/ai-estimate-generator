# AI 견적서 생성기

OpenAI API를 활용하여 전문적인 견적서를 자동으로 생성하는 웹 애플리케이션입니다.

## 주요 기능

- **두 가지 입력 방식 지원**
  - 구조화된 입력: 필드별로 체계적으로 입력
  - 원시 데이터 입력: 채팅 내역 같은 줄글 형태로 자유롭게 입력
- **AI 기반 견적서 생성**: OpenAI GPT-4o 모델 사용
- **실시간 미리보기**: 생성된 견적서를 즉시 확인
- **HTML 코드 편집**: 생성된 코드를 직접 수정 가능
- **PDF 다운로드**: 고품질 PDF 파일로 내보내기
- **템플릿 기반 생성**: 기존 템플릿을 활용한 일관된 견적서 생성
- **부분 교체 방식**: 전체 HTML 생성 대신 필요한 부분만 교체하여 효율성 향상

## 파일 구조

```
견적서 AI/
├── 견적서_AI_생성기.html    # 메인 애플리케이션
├── script.js               # JavaScript 로직
├── styles.css              # CSS 스타일
├── config.js               # API 키 설정 파일
├── .env                    # 환경 변수 (참고용)
├── fornerds_logo.png       # 로고 이미지
├── 견적서_템플릿.html       # 기본 견적서 템플릿
├── 견적서.html             # 원본 견적서 템플릿
├── 견적서_시안디자인_DB연동.html  # 예시 견적서
└── README.md               # 이 파일
```

## 설정 방법

### 1. API 키 설정

`config.js` 파일에서 OpenAI API 키를 설정하세요:

```javascript
const CONFIG = {
    OPENAI_API_KEY: 'your-api-key-here'
};
```

### 2. 웹 서버 실행

로컬 웹 서버를 실행하여 애플리케이션을 실행하세요:

```bash
# Python 3 사용
python -m http.server 8000

# 또는 Node.js 사용
npx http-server

# 또는 Live Server 확장 프로그램 사용 (VS Code)
```

### 3. 브라우저에서 접속

`http://localhost:8000/견적서_AI_생성기.html`로 접속하세요.

## 사용 방법

### 구조화된 입력 방식

1. "폼 입력" 탭 선택
2. 프로젝트 정보를 각 필드에 입력
3. "AI 견적서 생성" 버튼 클릭
4. 생성된 견적서 확인 및 편집
5. "PDF 다운로드" 버튼으로 최종 파일 저장

### 원시 데이터 입력 방식

1. "텍스트 입력" 탭 선택
2. 텍스트 영역에 프로젝트 정보를 자유롭게 입력
3. "AI 견적서 생성" 버튼 클릭
4. 생성된 견적서 확인 및 편집
5. "PDF 다운로드" 버튼으로 최종 파일 저장

### 파일 업로드 기능

- TXT, PDF, DOC, DOCX, MD 파일 업로드 지원
- 업로드된 파일 내용을 AI가 분석하여 견적서 생성에 활용
- 회의록, 요구사항 문서 등을 업로드하여 더 정확한 견적서 생성

## 커스터마이징

### 견적서 스타일 수정

`styles.css` 파일에서 스타일을 수정할 수 있습니다:

- 색상 테마 변경
- 폰트 스타일 수정
- 레이아웃 조정
- 견적서 템플릿 디자인 변경

### AI 프롬프트 수정

`script.js` 파일의 `systemPrompt` 변수를 수정하여 AI의 응답 방식을 조정할 수 있습니다.

### 템플릿 수정

`견적서_템플릿.html` 파일을 수정하여 기본 견적서 양식을 변경할 수 있습니다.

## 보안 주의사항

- `config.js` 파일에 API 키가 포함되어 있으므로 공개 저장소에 업로드하지 마세요
- 프로덕션 환경에서는 서버 사이드에서 API 키를 관리하는 것을 권장합니다
- `.env` 파일은 참고용이며, 실제로는 사용되지 않습니다

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **AI**: OpenAI GPT-4o-mini API
- **PDF 생성**: html2pdf.js
- **파일 처리**: PDF.js, Mammoth.js
- **코드 편집**: CodeMirror
- **스타일링**: CSS Grid, Flexbox

## 주요 특징

- **템플릿 기반 생성**: 일관된 견적서 양식 유지
- **부분 교체 방식**: 필요한 섹션만 교체하여 효율성 향상
- **실시간 미리보기**: 생성 과정을 실시간으로 확인
- **코드 편집**: HTML 코드를 직접 수정 가능
- **다양한 파일 형식 지원**: PDF, DOC, DOCX 등 다양한 파일 업로드
- **현실적인 개발 일정**: 앱 개발에 맞는 적절한 개발 기간 설정
